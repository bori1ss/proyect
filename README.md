Diseño preliminar del proyecto para que estudiantes y docentes organicen sus actividades académicas-

Requisitos: Python 3.10 o superior, VSCODE y la librería customtkinter.
Instalación:
En la terminal escribir:
'pip install customtkinter'

Ejecución:
En la terminal escribir:
'python main.py'


La base de datos `control_actividades.db` se crea automáticamente en la primera ejecución.
Archivos:
- `main.py` – punto de entrada de la aplicación.
- `database.py` – conexión y esquema de la base de datos.
- `auth_screen.py` – registro e inicio de sesión.
- `docente_screen.py` – lógica del rol Docente.
- `estudiante_screen.py` – lógica del rol Estudiante).
- `widgets.py` – componentes visuales reutilizables.
